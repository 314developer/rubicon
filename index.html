rubicon

Part of the BeeWare suite.

A collection of tools to bridge between Python and other language environments.

Rubicon libraries exist for:

  • Objective C
  • Java

Objective C

Rubicon-ObjC is a bridge between the Objective C libraries and Python. It has been tested on both desktop/laptops, and on iOS devices. It enables you to:

  • Use Python to instantiate objects defined in Objective C,
  • Use Python to invoke methods on objects defined in Objective C, and
  • Subclass and extend Objective C classes in Python.

It also includes wrappers of the some key data types from the Core Foundation framework (e.g., NSString and NSObject).

Java

Rubicon-Java is a bridge between the Java Runtime Environment and Python. It has been tested on both desktop/laptops, and on Android devices. It enables you to:

  • Instantiate objects defined in Java,
  • Invoke static and instance methods on objects defined in Java,
  • Access and modify static and instance fields on objects defined in Java, and
  • Write and use Python implementations of interfaces defined in Java.
